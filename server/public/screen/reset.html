🗑️ Reset Lab'O

Outil de réinitialisation pour repartir sur une base propre. À utiliser une seule fois.

⚠️ Ces opérations sont irréversibles. Toutes les données seront supprimées définitivement.

1. Réinitialiser la base de données (MongoDB)

Supprime tous les posts, annonces, coworkers, questions, alertes et médias.

2. Réinitialiser les données locales (navigateur)

Supprime les données IndexedDB stockées dans ce navigateur (annonces, partenaires, events, etc.).

3. Reset complet (MongoDB + IDB)
