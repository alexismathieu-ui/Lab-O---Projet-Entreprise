## 🚀 Installation & Setup

L'installation est conçue pour être "Zero-Config". Toutes les dépendances sont isolées dans des conteneurs.

### 1️⃣ Prérequis
* Avoir **Docker** installé sur votre machine (inclut Docker Compose).

### 2️⃣ Lancement
Ouvrez un terminal dans le dossier du projet et exécutez :

```bash
# Cloner le projet
git clone [https://github.com/alexismathieu-ui/Lab-O---Projet-Entreprise.git](https://github.com/alexismathieu-ui/Lab-O---Projet-Entreprise.git)

# Entrer dans le dossier
cd Lab-O---Projet-Entreprise

# Lancer l'infrastructure
docker compose up

3️⃣ Accès
Une fois le déploiement terminé, l'application est disponible sur :
👉 http://localhost:PORT (Remplacez PORT par le port configuré, ex: 3000)
.
├── docker-compose.yml    # Orchestration des services
├── src/                  # Code source de l'application
├── database/             # Scripts et volumes de données
