⚙️ Installation et Configuration
L'un des points forts de ce projet est sa facilité de mise en place. Vous n'avez pas besoin d'installer les dépendances individuellement sur votre machine.

Prérequis
Avoir Docker installé sur votre machine.

Avoir Docker Compose (inclus par défaut avec Docker Desktop).

Lancement du projet
Cloner le dépôt :

git clone https://github.com/alexismathieu-ui/Lab-O---Projet-Entreprise.git
cd Lab-O---Projet-Entreprise

Démarrer l'application :
Dans le dossier racine du projet, exécutez la commande suivante :

docker compose up

Accéder à l'application :
Une fois que les conteneurs sont lancés, ouvrez votre navigateur et rendez-vous sur :
http://localhost:[PORT] (Remplace [PORT] par le port utilisé par ton app, ex: 3000 ou 8080).
