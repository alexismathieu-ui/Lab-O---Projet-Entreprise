🚀 Lab'O — Projet Entreprise
📌 Description

Lab'O est un projet réalisé dans le cadre d’un projet d’entreprise.
L’objectif est de proposer une application permettant de mettre en avant des startups sous forme de présentation dynamique.

Le projet met l’accent sur :

L’affichage de contenus dynamiques
Une interface moderne
Une gestion côté admin pour contrôler les éléments affichés

⚙️ Installation & Lancement
Prérequis

Avant de commencer, assure-toi d’avoir installé :

Docker
Docker Compose
🚀 Lancer le projet
Clone le repository :
git clone https://github.com/alexismathieu-ui/Lab-O---Projet-Entreprise.git
cd Lab-O---Projet-Entreprise
Lance le projet avec Docker :
docker compose up

✅ Et voilà ! Le projet est maintenant lancé.

🎯 Fonctionnalités principales
📊 Affichage des startups

🧩 Architecture modulaire avec Docker
⚠️ Remarques
Les éléments affichés (comme les startups) ne disparaissent pas automatiquement sauf action de l’administrateur.
Le système de timer est utilisé uniquement pour le changement de diapositive, pas pour la suppression des données.
